## Sobre o Projeto

Este projeto foi desenvolvido como parte do **Trabalho Prático N1 da disciplina de Front End / Arquitetura Web**.

O objetivo do trabalho foi criar um **portal institucional estático e multi-página**, aplicando os conceitos fundamentais de **HTML5 semântico, CSS3 e Web Design Responsivo**.

O site desenvolvido representa uma loja chamada **StanleyKicks**, uma loja especializada na venda de tênis. O projeto apresenta uma página inicial com produtos, uma página sobre a empresa e uma página de contato.
